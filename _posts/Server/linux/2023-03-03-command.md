### 1. 기본 명령어 

#### 1-1. man : 메뉴얼 확인 
```
man [option] <command>
``` 

|옵션|내용|
|:---:|---|
|-a<img width=200/>|모든 페이지 출력<img width=750/>|
|-f|간단 설명만 출력|

<br>

#### 1-2. date : 시스템 시간/날짜 확인
```
date <MMDDhhmm[[CC]YY][.ss]>
```

|옵션|내용|
|------|---|
|MM|월|
|DD|일|
|hh|시간|
|mm|분|
|CC|연도 앞|
|YY|연도 뒤|
|ss|초|
